## Development

### Prerequisites

- [Node.js](https://nodejs.org/en/) (v12.16.1)

### Installation

```bash
npm install
```

### Running the app

```bash
npm start
```

### Building the app

```bash
npm run build
```

### Running the tests

```bash
npm run test
```

### Linting the app

```bash
npm run lint
```

### Formatting the app

```bash
npm run format
```

### Deploying the app

```bash
npm run deploy
```
